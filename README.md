# 广州旅行 AI 小助手 · 小粤

“小粤” 是专为广州旅行场景打造的 **RAG 增强型** AI 助手。通过集成检索增强生成技术，它能深度检索本地旅游知识库，提供比通用大模型更准确、更接地气的羊城攻略，有效解决 AI “幻觉”问题。支持多轮对话、实时工具调用、全栈开源！

---

## 🌟 核心技术：RAG 检索增强生成 (Retrieval-Augmented Generation)

本项目引入 RAG 技术作为核心引擎，确保建议的真实性与时效性：

- **精准知识库**：内置广州景点介绍、老字号美食、交通政策等 Markdown 格式文档。
- **智能文本处理**：利用 `RecursiveCharacterTextSplitter` 将文档切分为 500 字左右的小块，并设置 50 字重叠度以确保语义连贯。
- **向量化检索**：基于通义千问 `text-embedding-v1` 模型进行语义向量化，支持毫秒级的相似度搜索。
- **消除幻觉**：系统优先基于检索到的真实知识库内容进行生成，确保 AI 不再“一本正经地胡说八道”。

## 功能亮点

| 功能                    | 描述                                                         |
| :---------------------- | :----------------------------------------------------------- |
| 🧠 **RAG 增强问答**     | **基于本地知识库提供精准推荐，杜绝 AI 幻觉，支持小众攻略**   |
| 🗣️ **多轮对话**         | 流式输出（SSE），具备上下文记忆，支持 Markdown 格式渲染      |
| 🎭 **角色定义**         | 一键切换「本地通」、「美食家」、「导游」等定制化人格         |
| 🛠️ **Function Calling** | **自动触发工具**：实时天气查询、高铁余票检索及本地知识库搜索 |
| 🎯 **回答范围限制**     | 深度约束 AI 仅回答广州旅行相关话题，拒绝无关跑题             |
| 💾 **对话持久化**       | 当前对话自动保存，支持历史记录随时回顾                       |
| 🌦️ **实时出行查询**     | 提供广州实时天气、空气质量、穿衣建议及全国往返高铁余票       |
| 🛠️ **投诉通道**         | 模拟真实政务场景，支持文字快速提交并由后台自动分类派单       |

## 🛠️ 技术栈

| 层级         | 技术实现                                             |
| :----------- | :--------------------------------------------------- |
| **前端**     | Vue3 + Pinia + Vant + Fetch (SSE 流式解析)           |
| **后端**     | Node.js + Koa2 + **LangChain.js** (RAG 核心编排框架) |
| **向量存储** | `MemoryVectorStore` (基于内存的高速向量数据库)       |
| **AI 引擎**  | DeepSeek API + 通义千问 (Embedding & LLM)            |

## 📂 快速上手 (RAG 模式)

1. **知识库准备**：将最新的广州旅游攻略（Markdown 格式）放入 `knowledge/` 目录。
2. **索引构建**：系统启动时会自动读取文档并调用 Embedding 模型完成向量化存储。
3. **智能交互**：提问“广州塔怎么去”或“东山口有什么好吃的”，AI 将自动检索知识库并给出详细指引。
