# 广州旅行 AI 小助手 · 小粤

“小粤” 是专为广州旅行场景打造的 AI 助手，支持多轮对话、实时天气/车票查询、智能投诉及角色扮演，前后端全栈开源！

## 功能亮点

| 功能                  | 描述                        |
| ------------------- | ------------------------- |
| 🗣️ **多轮对话**        | 流式输出，上下文记忆，支持 Markdown 渲染 |
| 🎭 **角色定义**         | 一键切换「本地通」、「美食家」、「导游」等角色   |
| 🎯 **回答范围限制**       | 只聊广州旅行相关，拒绝跑题             |
| 💾 **对话保存**         | 当前对话自动保存，随时回顾            |
| 🌦️ **天气查询**        | 实时广州天气、空气质量、穿衣建议          |
| 🚄 **车票查询**         | 广州⇄全国高铁/普列余票          |
| 🛠️ **投诉通道**        | 文字快速提交，后台自动分类派单        |

## 技术栈

| 层级        | 技术                                       |
| --------- | ---------------------------------------- |
| **前端**    | TypeScript + Vue3 + Pinia + Vant + Fetch |
| **后端**    | Node.js + Koa2                           |
| **AI 引擎** | DeepSeek API + 通义千问 API                  |
